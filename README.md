# Anish Patankar - Portfolio

## Getting Started

This is a portfolio website built with React, TypeScript, and Tailwind CSS.

Clone the repository and follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd anish-portfolio

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy using your preferred hosting service (Vercel, Netlify, GitHub Pages, etc).

To deploy:
- Build the project: `npm run build`
- Deploy the `dist` folder to your hosting service

## Custom Domain

You can connect a custom domain through your hosting provider's dashboard.
